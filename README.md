Features:
* Data type: integer and boolean
* Variable
* Arithmethic Operations
* Comparison Operator
* Conditional
* Loop
* Array
* Function
* Console Input
* Console Output
* Scoped Expression
* Static Single Assignment Intermediate Representation
* Register Allocation by Graph-Coloring
